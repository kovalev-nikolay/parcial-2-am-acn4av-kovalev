# parcial-1-am-acn4av-kovalev
Aplicación móvil para registrar y analizar lanzamientos de baloncesto durante entrenamientos.
## 📱 Funcionalidad

- Visualización de estadísticas de lanzamientos
- Porcentaje de aciertos (UI interactiva)
- Diferentes tipos de tiros (libres, campo, triples)

## 🛠 Tecnologías

- Java
- Android Studio
- XML layouts
